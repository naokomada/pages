本を開く前の経験

本を開く前の段階で、読書の経験を始まっている。本が陳列されている場合、最初に。目にされるのは背表紙か、もしくは表紙である。背表紙から得られる本の情報はほんのわずかである。にも関わらず、我々はそこから本の内容を想像する。想像ができるが、正確にその中身を知ることはできない。本に書かれていることは、こことは別の空間のことか、別の時間のことである。このため、本の背表紙は別の空間や時間に対する裂け目のように存在している。本が陳列されている場所というのは、別の世界へアクセスするための裂け目がたくさん開かれている場所だということができる。

一方で、世界の中には本が陳列された場所がたくさんある。
