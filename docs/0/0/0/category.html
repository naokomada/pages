カテゴリについて

人間の認知能力では、同時に把握ができる数は限られている。このため人間が多くの存在物を把握するためには、そもそも同時に認知するものの数を減らすことが必要になる。
カテゴリーはこのために存在する。

カテゴリの数

プラトンのカテゴリは5つであった。

アリストテレスは述語の種類を10に区別した。また存在者を10に分類した。

あらゆるものを限られた数に分類するために、10のカテゴリに分ける方法がある。10は人間の人体における指の数に由来している可能性がある。また、十進法との相性がよく、複数の階層をもつ分類法においても、各階層のカテゴリが10づつに分かたれていると、階層と同じ桁数の数値によって各カテゴリを表現できる。
